# System Design Interview Prep

63 system design questions with full speak-aloud playbooks, Mermaid architecture diagrams, and quick-reference cheat sheets. Targeting senior / team-lead level interviews.

Each folder contains:
- **README.md** — 17-section speak-aloud playbook (~300-400 lines)
- **architecture_diagram.md** — Mermaid diagrams (high-level + deep-dive + sequence)
- **cheat_sheet.md** — One-page quick-reference card (~40 lines)
